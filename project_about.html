How much can we learn about a given geographic region based on the content of its corresponding social media posts (Twitter)?

In my project I develop a web app to help do this by analyzing upwards of two million tweets collected from users in several major United States cities in late 2014.

I fit a mixture model (details below) to the historical geocoded tweet data in a city and allow a user-specified set of keywords to guide the prior probabilities of each tweet belonging to a particular component.

After fitting the model my tool can be used to profile a city and see where people tend to congregate and tweet about certain topics.

Why don't you try it yourself?

Details of the Mixture Model for Geocoded Tweets

We can easily plot a geocoded tweet as a point on a map using its longitude and latitude coordinates (left). Each tweet also contains a small set of words, so in this way we can also visualize where people post about particular topics (right).

A key assumption in my mixture model is that there will be one component of random noise throughout a city, and a number of other components that have gaussian distributions with relatively small variances. I allow the presence of a word, e.g. "beer", in a tweet to affect the prior probability of its spatial coordinates belonging to one of the gaussian components (left). After using an EM algorithm to fit this model, you can use the estimated locations of the components (right) to say something informative about the behavior of social media users in a particular city with respect to those keywords.
